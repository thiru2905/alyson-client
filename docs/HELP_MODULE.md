# Help Module Documentation

## Purpose

Help (`/help`) is a static, searchable FAQ covering major Alyson HR modules. No backend — content lives in the route file.

## Route

- **File:** `src/routes/help.tsx`
- **Path:** `/help`

## Topics Covered

- Getting started
- Alyson Notetaker
- Meeting calendar
- Handover docs
- Workspace activity
- Time dashboard
- Payroll, equity
- Contact support (`thirumalai@cintara.ai`)

## Key Behaviors

- Searchable topic list
- Expandable cards + detail FAQ panel
- Links/references to other modules and S3 paths

## Access

All authenticated roles
